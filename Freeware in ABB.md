# Freeware in ABB

ABB provides employees with access to a wide range of approved freeware applications through a centralized platform called the **Company Portal**.

## Available Categories

The Company Portal includes various categories of freeware, such as:

- ABB Applications
- Business Tools
- Development Tools
- Productivity Tools
- Other approved software categories

## Installation Process

- Installation of approved freeware does **not require any additional approval**.
- Users can directly access the Company Portal.
- Search for the required application.
- Download and install the software as needed.

## Software Updates

- The global team periodically updates the Company Portal.
- New authorized freeware applications are regularly added and made available to users.

---

## Global Software Availability Through Company Portal

The global team is also planning to include selected global software products within the Company Portal, such as:

- AutoCAD
- Selected Microsoft Products
- Other approved global software

This will enable users to:

- Download and install software directly from the Company Portal.
- Avoid raising a MyIS ticket for eligible software installations.
- Access software through a simplified self-service process.

---

## Software Charges

Some software available through the Company Portal may be chargeable.

### Important Notes

- The applicable cost will be clearly displayed during the installation process.
- By proceeding with the installation, the user acknowledges the associated charges.
- Charges will continue to be recovered through the standard **CHGPL invoicing process**.
