# ABB India Limited - Software Licensing

## Overview

ABB India Limited has a **100+ year presence** with a unique confluence of Business, GBS, and AIC operations.

### Key Facts

- 5 Manufacturing Locations
- 25 Manufacturing Plants
- 21 Sales Offices
- 4 Main Divisions
- 8000+ Employees

ABB has **Zero Tolerance** towards non-compliance and expects all employees to adhere to internal policies and procedures regarding sourcing, usage, sales, and services.


# Software Licensing in ABB

There are two types of license procurement:

1. Global License
2. Local License

# Global Licenses (Through MyIS)

### Products Covered

- Microsoft
- Oracle
- Autodesk
- Baan
- SAP
- PTC
- Adobe
- SolidWorks

### Guidelines

- Products must be procured **only through CHGPL (MyIS)**.
- Licenses are to be used **only on ABB assets**.
- Usage will be reported in the **Global License Report (GLR)**.
- Software repository is maintained in the local FTP (access provided to Wipro).
- Do **not** source software outside CHGPL.
- **Exception:** Procurement outside CHGPL is allowed only if the software is unavailable in the global pool and approved by the CIO and Group IS.

# Local Licenses (Through Local SCM)

### Procurement Process

- All third-party software must be sourced from **authorized software vendors only** through IS.
- IS will provide quotations and share negotiated/discounted prices with Business Areas (BAs).
- BA/Function teams must raise PR/PO through the GBS SCM team and share the PO with IS.
- IS will forward the PO to the vendor and act as an interface between the vendor and the requesting BA/Function for:
  - Procurement
  - Renewal
  - Agreements
  - Other software-related activities

# IS Charges Portal

The **IS Charges Portal** enables users to view their software usage and installation reports.

> Reported usage is limited to software purchased by ABB under Global Contracts.

A complete list of globally contracted software can be found in the **License Pricing** section of the portal.

### Ownership and Responsibilities

- OSM owns and administers the database.
- Content responsibility lies with the respective data owners.
- Software usage reports and country reporting structures are managed by:
  - Country IS Teams
  - SAM Managers
  - Global Vendor Managers

# Benefits of IS Charges Portal

The IS Charges Portal:

- Helps manage software assets effectively.
- Provides financial control over license pools.
- Serves as a source of data for:
  - Direct software charges
  - UBC charges
  - IT IS charge allocations
- Assists in planning budgets for future years.

# Software Requests and Incident Management

All global software products can be sourced through **MyIS (ServiceNow)** by submitting a **Request**.

Some software can also be sourced through the **Company Portal**, including:

- Authorized freeware
- Software that can be installed without line manager approval

### Support Process

For issues related to software or applications:

- Raise an **Incident Ticket (INC)** in MyIS.
- Service Desk is responsible for handling:
  - Request Tickets (RITM)
  - Incident Tickets (INC)

All requests and incidents are fulfilled according to the defined SLA timelines.

# Software Charges

All software listed in the **IS Charges Portal** is chargeable.

### Billing

- Charges are billed through the **CHGPL Invoice**.
- Applicable software usage costs will be allocated accordingly.
