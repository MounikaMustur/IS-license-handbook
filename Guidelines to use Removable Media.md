# Guidelines for Using Removable Media (USB / Hard Disk) in ABB

ABB provides clear and unambiguous guidance to employees regarding the security measures that must be followed when using removable media devices.

## Purpose

The use of removable media, such as USB drives and external hard disks, should be limited to situations where it is the only practical or approved method for transferring information between different systems.

## Security Considerations

When using removable media, employees must:

- Follow ABB Information Security policies and procedures.
- Ensure that removable media is used only for legitimate business purposes.
- Protect company information from unauthorized access, loss, or disclosure.
- Comply with all applicable security controls and data protection requirements.
- Verify that the removable media device is authorized and secure before use.

This document provides guidance on:

- Safe handling of USB drives and external storage devices.
- Information security best practices.
- Data transfer procedures.
- Risk mitigation measures.
- Compliance requirements for removable media usage.

## Compliance

All employees are expected to adhere to the prescribed guidelines whenever removable media is used to transfer or store ABB information.

Failure to comply with these guidelines may result in security risks and non-compliance with ABB Information Security policies.
