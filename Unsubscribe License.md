# How to Unsubscribe Unused License

End users can unsubscribe unused software licenses through a simple **one-click solution** available in the **IS Charges Review Portal**.

## Process Overview

- Users can log in to the **IS Charges Review Portal**.
- Select the unused software license.
- Submit an unsubscribe request with a single click.

## Billing Impact

- Once unsubscribed, the software will **not be charged from the corresponding quarter of the year**.
- This ensures cost optimization and better license management.

## Approval Requirement

- The unsubscribe request must be approved by the **concerned software owner**.
- Only after approval will the user receive the benefit of charge removal.

## Communication

- Periodic email notifications will be sent to users.
- These emails will remind users to review and unsubscribe unused licenses where applicable.

## Portal Link

- [ISC Review Portal](https://review.ischarges.abb.com/)
