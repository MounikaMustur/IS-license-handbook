# Authorized Software List in ABB

## Whitelisted Third-Party Software

ABB maintains a list of approved and authorized third-party software applications, commonly referred to as the [Authorized Software List](https://app.powerbi.com/groups/me/reports/7756b2c8-a186-4b1d-9cbe-f1a17729b190/ReportSectioncabce2d3adc0bd07a0a6?experience=power-bi).

Before installing or using any software, users should verify whether the software is authorized for use within ABB.

### Verification Process

1. Check the **Authorized Software List** using the provided organizational link.
2. Confirm whether the software is approved for use within ABB.
3. For additional clarification or confirmation, contact the **Country Software Asset Manager (Country SAM)**.

---

## User Responsibilities

End users are required to adhere to ABB organizational policies and procedures regarding:

- Software installation
- Folder access
- System access permissions
- Usage of third-party applications
- Compliance with software licensing requirements

---

## Compliance Monitoring

ABB Information Security conducts periodic compliance monitoring across all corporate assets connected to the ABB network.

### Monitoring Activities

- Automated scripts are executed every fortnight.
- Assets are scanned for:
  - Unauthorized software installations
  - Unauthorized access permissions
  - Non-compliant software usage

### Notification Process

- Users identified with unauthorized installations or access will receive an automated notification email.
- The notification will provide details of the detected software or access concern.
- Country SAMs may assist users in understanding and resolving compliance issues.

### Awareness and Support

The compliance process helps:

- Increase awareness of authorized software usage.
- Reduce software licensing and security risks.
- Ensure adherence to ABB policies and standards.
- Support users through guidance provided by Country SAMs.

---

## User Response Requirement

Upon receiving a compliance notification email, users are required to:

- Confirm the installation or access if it is legitimate and authorized.
- Reject or report the installation/access if it is incorrect or no longer required.
- Provide any necessary justification or information requested in the communication.

Timely responses help maintain software compliance and ensure accurate asset management across ABB.
